# 🚀 Awax

Projeto desenvolvido com HTML e CSS como prática de desenvolvimento front-end, recriando o layout da landing page Awax com foco em responsividade e organização de código.

## 📋 Sobre o Projeto

O Awax é uma landing page moderna desenvolvida para aprimorar conhecimentos em:

* Estruturação semântica com HTML5
* Estilização com CSS3
* Flexbox
* Responsividade
* Organização de componentes visuais
* Boas práticas de desenvolvimento front-end

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3

## ✨ Funcionalidades

* Header com menu de navegação
* Banner principal
* Seção de serviços
* Seção sobre a empresa
* Portfólio de projetos
* Área de equipe
* Depoimentos
* Formulário de contato
* Footer responsivo

## 📱 Responsividade

O projeto foi desenvolvido para adaptação em diferentes tamanhos de tela:

* Desktop
* Tablet
* Smartphone

## 🎯 Objetivo

Este projeto foi criado com fins de estudo e aprimoramento das habilidades em desenvolvimento front-end, praticando a construção de interfaces modernas e responsivas.
